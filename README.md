# Symfony Docker Compose

Symfony 4 project running into Docker containers:
- Php:7.4-fpm
- Nginx:1.19.0-alpine
- MySql:8.0
- Adminer:lastest

### Install Docker

```bash
apt-get install docker docker-compose
```
### Run

```bash
docker-compose up
```
```bash
docker-compose exec php composer install 
```
### Go To 
- http://localhost/
